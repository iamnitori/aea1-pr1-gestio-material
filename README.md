# Sistema de gestió de material
0487 - AEA1 - Pràctica 1  
Nil Torrens Ripoll  

## Descripció
Aquest projecte és una primera versió d’un sistema de gestió de material per a un treball de classe. Permet gestionar inventari, préstecs i categories de material de manera bàsica.  
Encara que no inclou totes les funcionalitats completes, aquest projecte proporciona la base per al desenvolupament futur i inclou la documentació necessària per a usuaris i programadors.  

## Contingut del repositori
- `src/` : Codi font Java del projecte.
- `doc/` : Documentació generada amb Javadoc.
- `Documentació d’usuari.pdf` : Guia per a professors i personal del centre sobre l’ús del sistema.
- `Documentació tècnica.pdf` : Explicació de l’arquitectura, classes i instruccions per a desenvolupadors.
